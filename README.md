# Projeto Fitness 

Sistema completo para ingestão alimentar, cálculo de calorias (USDA API), TMB (Harris-Benedict), GET, relatório e salvamento no Azure Blob Storage.

## Tecnologias:
- Azure Function (Python)
- Azure Web App + Streamlit
- Azure Blob Storage
- GitHub Actions (CI/CD)
- Docker

## Pastas:
- `azure-function/` — backend
- `web/` — frontend streamlit
- `infra/` — infraestrutura
- `docs/` — documentação
